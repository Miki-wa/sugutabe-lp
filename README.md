# sugutabe-lp
架空のオーガニック食品サービス「スグ食べ」のランディングページです。

## URL
https://miki-wa.github.io/sugutabe-lp/

## 概要
スクール課題として、ヒアリング内容をもとに制作したランディングページです。  
リスティング広告からの流入を想定し、サービスの魅力が伝わる構成を意識しました。

## 目的
食にこだわりのあるユーザーに向けてサービスの魅力を伝え、お試し購入へ導くこと

## ターゲット
食の安全や品質に関心が高く、「ていねいな暮らし」を大切にする層

## 使用技術
- HTML
- CSS（レスポンシブ対応）

## 工夫した点
- 「新鮮さ・安心感・信頼感」が伝わるよう、ナチュラルで清潔感のあるデザインを意識
- ファーストビューでサービスの特徴が伝わる構成
- CTAボタンを目立つ位置に配置し、行動を促す導線を設計
- スマートフォンでの閲覧を意識した視認性・操作性への配慮

## 改善案
- 利用者の評価（★など）や具体的な利用シーンを追加し、信頼性の向上
- CTAの配置やデザインの検証・改善によるコンバージョン率向上
